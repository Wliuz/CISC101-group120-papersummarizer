There are a total of 8 files each containing specific rules and expectations for the system prompt. the reference framework is to show the the expected output would be if the LLM were to follow the system prompt. furthermore, the intake and setup involved all the required inputs. the plan builder are constraints the LLM must follow. the guardrails are constraints and if statements for the LLM to follow if it runs into any of thiss scenarios. moduke 4 is for refining the output and making sure it follows consistent formatting. finally the last 2 modules, citation extractor involves citing information and definition extractor involves extracting definitions of expert terms for laypeople to understand.
